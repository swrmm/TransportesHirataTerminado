RF-06: Registrar mantenimiento de equipos de oficina

Objetivo:
Permitir que un tecnico registre mantenimientos preventivos o correctivos de equipos de oficina, como PCs, impresoras u otros equipos registrados.

Actor:
Personal de mantenimiento. En el sistema corresponde al usuario con cargo "personal".

Autenticacion:

El tecnico ingresa al sistema desde el login con su RUT y clave.

El login valida los datos en la tabla Usuario.

Si el usuario tiene cargo "personal", puede entrar al menu Mantenimiento.

El Personal solo puede registrar mantenimientos de equipos de oficina.

No puede realizar mantenimientos de camiones, ver el historial de camiones ni actualizar kilometraje.

Usuario tecnico de prueba:
RUT: 2.222.222-2
Clave: 12345678
Cargo: personal

Registro de nuevos usuarios de personal:
Solo el administrador puede crear usuarios de personal.
Para hacerlo debe entrar a Registro -> Registrar personal.
Los usuarios creados ahi quedan con cargo "personal".

Flujo normal:
1. El tecnico inicia sesion.
2. Entra al menu Mantenimiento.
3. Selecciona la opcion "Hacer mantenimiento equipo".
4. El sistema abre la ventana de mantenimiento de equipos.
5. El tecnico selecciona un equipo existente desde la tabla.
6. El sistema carga automaticamente el ID, el equipo y la fecha actual.
7. El tecnico completa:
   - Tipo de mantenimiento: preventivo o correctivo.
   - Descripcion del trabajo realizado.
   - Tecnico responsable.
   - Observaciones.
   - Estado: realizado o pendiente.
8. Presiona el boton Guardar.
9. El sistema guarda el mantenimiento en la tabla MantenimientoEquipo.

Validaciones:
- Si no hay equipos registrados, el sistema avisa que primero debe registrarse un equipo.
- Si no se selecciona un equipo, el sistema solicita seleccionar uno.
- Si falta descripcion o tecnico responsable, el sistema muestra un error.
- Si el equipo seleccionado no existe en la base de datos, el sistema avisa que debe registrarse primero.

Base de datos:
El mantenimiento se guarda en la tabla MantenimientoEquipo.
Esta tabla se relaciona con la tabla equipos mediante idEquipo.

Resultado:
El mantenimiento queda registrado correctamente y asociado al equipo seleccionado.
Este registro queda disponible para usarse despues en un historial de mantenimiento.
