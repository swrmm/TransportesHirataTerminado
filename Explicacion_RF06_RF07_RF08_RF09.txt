EXPLICACION DE REQUERIMIENTOS FUNCIONALES AGREGADOS
Proyecto: Transportes Hirata

Este documento resume las cuatro funcionalidades agregadas en esta etapa del proyecto:
RF06, RF07, RF08 y RF09. Todas estan relacionadas con el control de equipos de oficina,
su mantenimiento, el software instalado y el inventario de piezas.


RF06 - Registro y mantenimiento de equipos de oficina
-----------------------------------------------------
Esta funcionalidad permite administrar los equipos de oficina usados por la empresa.

El administrador puede registrar equipos indicando datos como tipo, codigo, marca,
ubicacion y estado. Para mantener un registro ordenado, el tipo de equipo se selecciona
desde una lista con tres opciones:

- Computador
- Impresora
- Telefono

El codigo del equipo se genera automaticamente segun el tipo seleccionado. Por ejemplo:

- Computador: PC-001, PC-002, PC-003
- Impresora: IMP-001, IMP-002, IMP-003
- Telefono: TEL-001, TEL-002, TEL-003

Esta RF tambien permite registrar mantenimientos asociados a un equipo existente.
El sistema guarda fecha, tipo de mantenimiento, descripcion, tecnico responsable,
observaciones y estado del mantenimiento.

Objetivo:
Mantener un control claro de los equipos de oficina y registrar las acciones de
mantenimiento que se realizan sobre ellos.


RF07 - Actualizacion de software en equipos
-------------------------------------------
Esta funcionalidad permite registrar actualizaciones de software realizadas en los
equipos de oficina.

El usuario puede seleccionar un equipo y un software desde el catalogo registrado.
Luego puede ingresar la version anterior, version nueva, tecnico responsable, fecha,
estado y observaciones.

Si el software no existe en el catalogo, el sistema permite registrarlo primero con
datos como nombre, version actual, proveedor, tipo y estado.

Cuando una actualizacion queda marcada como realizada, el sistema actualiza la version
instalada para ese equipo y tambien actualiza la version actual del software en el
catalogo.

Objetivo:
Controlar que software se instala o actualiza en cada equipo, mantener historial de
cambios y evitar registros manuales desordenados.


RF08 - Historial de mantenimiento de equipos
--------------------------------------------
Esta funcionalidad permite consultar el historial de mantenimientos realizados a los
equipos de oficina.

El sistema muestra una tabla con los mantenimientos registrados, incluyendo datos como:

- ID del mantenimiento
- Fecha
- Equipo
- Tipo de mantenimiento
- Descripcion
- Tecnico responsable
- Observaciones
- Estado

Tambien permite seleccionar un mantenimiento para revisar su informacion con mayor
claridad en el formulario de detalle.

Objetivo:
Permitir trazabilidad sobre los mantenimientos realizados, saber cuando se intervino
un equipo, quien lo hizo y cual fue el resultado.


RF09 - Control de inventario de piezas
--------------------------------------
Esta funcionalidad permite administrar piezas o repuestos usados en los mantenimientos
de equipos.

El administrador puede registrar piezas indicando codigo, nombre, categoria, marca,
stock actual, stock minimo, ubicacion y estado.

El sistema valida que no se registren dos piezas con el mismo codigo. Si una pieza ya
existe, se muestra una advertencia para evitar duplicidad.

Ademas, se pueden registrar movimientos de inventario:

- Entrada: aumenta el stock.
- Salida: descuenta stock.
- Ajuste: fija el stock en una cantidad especifica.

La RF09 tambien se integra con el mantenimiento de equipos. Al registrar un mantenimiento,
el usuario puede seleccionar una pieza de inventario y una cantidad utilizada. Si hay
stock suficiente, el sistema registra el mantenimiento y descuenta automaticamente la
pieza del inventario.

El estado de la pieza se actualiza segun el stock:

- disponible
- stock bajo
- sin stock

Objetivo:
Controlar las piezas disponibles, evitar uso de repuestos sin stock y relacionar el
inventario con los mantenimientos reales realizados sobre los equipos.


RELACION ENTRE LAS 4 RF
-----------------------
Las cuatro funcionalidades trabajan juntas:

1. RF06 registra los equipos y sus mantenimientos.
2. RF08 muestra el historial de esos mantenimientos.
3. RF07 controla el software instalado o actualizado en esos equipos.
4. RF09 controla las piezas usadas cuando un mantenimiento requiere repuestos.

En conjunto, estas RF agregan un modulo completo de administracion de equipos de oficina,
permitiendo controlar hardware, software, mantenimiento, historial e inventario asociado.
